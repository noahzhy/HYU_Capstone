# Add ShuffleNet

## 使用 ShuffleNet
```python
model = ShuffleTrackNet(cfg=cfg_shuffle).cuda()
#配置使用cfg_shuffle即可
```

## 训练

使用 Shuffledataset 训练构建数据集，参数为数据集图片所在路径，图片默认为jpg格式，需要其标签在同一目录下，标签文件为txt格式，要求与图片同名。例如test.jpg对应的标签文件为test.txt。之后使用train训练。

## 预测

使用 predict 进行预测。每次输入单张预测图片路径，以及输出路径
